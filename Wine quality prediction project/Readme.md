# 🍷 Wine Quality Prediction using Logistic Regression

## 📌 Project Overview
This project focuses on predicting the quality of red wine using Machine Learning.  
We use **Logistic Regression** to classify wine into:

- ✅ Good Wine (1)
- ❌ Bad Wine (0)

The prediction is based on physicochemical properties such as acidity, alcohol content, pH, etc.

---

## 🎯 Objective
To build a classification model that can determine whether a wine is of good quality or not based on its features.

---

## 📊 Dataset Information
- Dataset: Wine Quality (Red Wine)
- Source: UCI Machine Learning Repository
- Number of Features: 11
- Target Variable: Quality (converted to binary classification)

### Features Include:
- Fixed Acidity  
- Volatile Acidity  
- Citric Acid  
- Residual Sugar  
- Chlorides  
- Free Sulfur Dioxide  
- Total Sulfur Dioxide  
- Density  
- pH  
- Sulphates  
- Alcohol
